# CartoonFight: Советская Заруба

Добро пожаловать в **CartoonFight** — файтинг в стилистике 80-х, где на арене сходятся легенды советской мультипликации! Забудьте о дружелюбии: Волк из «Ну, погоди!» и Кот Леопольд решили выяснить, кто из них настоящий хозяин экрана.

## 🥊 Особенности игры

- **Ностальгический сеттинг:** Атмосфера советских мультфильмов, перенесенная в формат сурового файтинга.
- **Классическое противостояние:** Играйте за Волка или Леопольда.
- **Стиль 80-х:** Олдскульный геймплей с акцентом на тайминги и тактику.

## ⚔️ Игровые механики

### Атаки
- **Легкая атака:** Быстрый удар с коротким временем восстановления. Идеально подходит для прерывания действий противника.
- **Тяжелая атака:** Медленный, но мощный удар. Наносит значительный урон и может откинуть противника.

### Защита и Блок
- **Блок:** Удерживайте клавишу блока, чтобы снизить входящий урон на 50%.
- **Замедление:** Во время блока скорость передвижения персонажа снижается.

### Система оглушения
В игре реализована механика накопления безответных ударов:
- Если вы нанесете **3 удара подряд**, и противник не успеет ответить, он будет **оглушен на 3 секунды**.
- Оглушенный персонаж не может двигаться, но всё еще может пытаться огрызаться атаками в надежде сбросить счетчик.
- Ответный удар по нападающему сбрасывает накопленный счетчик ударов.

### Откидывание
Сильные удары физически отталкивают противника, что позволяет разрывать дистанцию или прижимать оппонента к краю арены.

## 🎮 Управление

По умолчанию управление настроено для двух игроков (настраивается в `PlayerControls`):

| Действие | Клавиша             |
| :--- |:--------------------|
| Движение влево | `A` / `Left Arrow`  |
| Движение вправо | `D` / `Right Arrow` |
| Легкая атака | `G` / `K`           |
| Тяжелая атака | `H` / `L`           |
| Блок | `Space` / `N`       |

## ⚙️ Техническая информация

- **Движок:** Unity 6000.4.1f1
- **Платформа:** PC
- **Система ввода:** Стандартный Input Manager
